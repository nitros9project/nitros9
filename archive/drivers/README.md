# Archived Drivers

These third-party driver packages were formerly located in
`3rdparty/drivers`.

They are preserved as historical source and are excluded from the normal
NitrOS-9 build. Some packages duplicate maintained drivers elsewhere in the
repository, while others support uncommon or obsolete hardware. Individual
package makefiles remain available for manual investigation, but successful
assembly and operation are not guaranteed.

The archive currently contains:

- Burke & Burke WD1002/WX1/WX2 hard disk drivers
- Compusense DragonPlus support
- Disto controller, RAM Pak, and parallel-port drivers
- emulator disk descriptors
- legacy IDE and MMC drivers
- NoCan RAM disk support
- legacy 16550 serial support
- D.P. Johnson and Sardis floppy drivers
- legacy TC^3 hard disk and SCSI support
