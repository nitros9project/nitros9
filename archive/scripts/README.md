# Archived scripts

These scripts document former release-distribution and project-publishing
workflows. They depend on directory layouts, products, or hosting services that
are no longer part of the supported repository.

- [`distribution/`](distribution/README.md) contains obsolete disk indexing,
  bundle-building, and boot/track list tools.
- [`publishing/`](publishing/README.md) contains the former SourceForge upload
  helper.

They are preserved for historical reference and are not invoked by supported
builds or CI.
