# Project history

These files document earlier NitrOS-9 release and repository eras. They are
retained as historical records rather than current project instructions.

- `ChangeLog` is the migrated historical change log.
- `distribution-readme.txt` describes legacy prebuilt floppy distributions.
- `release-todo.txt` records goals for releases from the 2000s; it is not the
  current issue tracker.
- `hgtags` preserves tags from the former Mercurial repository.

Current build and contribution instructions are in the repository root
[`README.md`](../../README.md) and [`CONTRIBUTING.md`](../../CONTRIBUTING.md).
