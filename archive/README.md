# NitrOS-9 Archive

This directory preserves historical NitrOS-9-related source code that is no
longer part of the maintained, default build.

Archived code may be incomplete, duplicated by newer implementations, tied to
obsolete hardware, or subject to licensing terms that differ from the rest of
the repository. It is retained for historical reference and possible manual
use.

Nothing under this directory is built by the top-level NitrOS-9 makefiles.

The archive currently includes:

- third-party device drivers
- third-party file managers
- experimental and demonstration system modules
- third-party subroutine modules
- historical utilities
- VEF image assets
- unfinished work-in-progress sources
- historical application packages
