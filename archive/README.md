# NitrOS-9 archive

This directory preserves historical NitrOS-9-related source, assets, and
project records that are not part of the maintained default build.

Archived material may be incomplete, duplicated by newer implementations,
tied to obsolete hardware, or subject to licensing terms that differ from the
rest of the repository. It is retained for historical reference and possible
manual investigation.

Nothing under this directory is built by supported recipes or CI. See the
[archive manifest](MANIFEST.md) for category and status information.
