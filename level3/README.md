# NitrOS-9 Level 3

This directory contains experimental Level 3 work for the CoCo 3 in 6809 and
6309 configurations. It explores mapping file managers and their local memory
into separate regions to relieve Level 2 system-map pressure.

Level 3 is not part of the supported recipe or continuous-integration matrix.
The source and setup material are retained for technical study and possible
future development; their presence does not imply a currently bootable or
supported distribution.

Historical implementation notes are in [`Level3.doc`](Level3.doc) and
[`ReadMe.l3`](ReadMe.l3). Those documents describe strict boot-module ordering
and other constraints that apply to the original experiment.
